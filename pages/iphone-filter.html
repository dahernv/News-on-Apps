How to use filter on Apps

You can tap on the search icon on the top right corner.
You will find the filter icon on the top right corner.
Once you tap on it, it will display different options.
You can filter the event by "Topics", like Diwali, Krishna, Ganesha, Durga, Workshop,etc. Once you choose a Topic, tap on "Done" on the top right corner.
Suppose we select "Diwali". As a result, you will get all the Diwali pujas.
Now imagine you want them in "Ascending date" or "Descending date" order. Go again to the filter and select the option "Sort by".
So you will get all the Diwali Pujas by ascending date.
You can add one more filters, like "Country".
And, as result, you will get all the Diwali Pujas, in the country selected by ascending date. In this example, we chose Italy.
Now, maybe you would like to share some of these events with your collective. So, you might want to have the subtitles in your language. Then, you can go to "Subtitles" and select the language.
As a result, you will get all the Diwali Pujas, in Italy, sorted by ascending date with your chosen subtitles, in the example we used Spanish.
Of course, because this is an example to show the filters and functionality, we selected one by one. But you can select all the filters at once and get the same result.
